Part 2: The "Principle of Least Privilege"

Standard Users (Officers)

✅ Browse web, Office, print
❌ Install software, change network settings, disable antivirus

Admin Accounts (IT)

Elevated privileges — only for installing, configuring, troubleshooting.
Never given to regular users.

Security Golden Rule: Users should have absolute minimum permissions needed for daily job. Prevents malware, accidental damage, and unauthorized changes.
