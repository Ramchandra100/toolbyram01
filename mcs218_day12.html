📡 1. Multiple Access: Pure ALOHA vs Slotted ALOHA

Random access protocols — throughput, vulnerable period, efficiency

📖 Explain

Pure ALOHA: Transmit anytime; if collision, wait random time & retransmit. Throughput S = G·e-2G, max ≈ 18% at G=0.5. Vulnerable period = 2T (T = frame time).

Slotted ALOHA: Time divided into slots; transmit only at slot start. Throughput S = G·e-G, max ≈ 36% at G=1. Vulnerable period = T.

📢 Analogy: Pure ALOHA = shouting anytime; Slotted ALOHA = shouting only on metronome ticks.

🎧 2. CSMA & CSMA/CD (Carrier Sense Multiple Access)

Listen before talk • Collision Detection • Binary Exponential Backoff

📖 Explain

CSMA: Stations listen before transmitting. If channel busy, wait. Reduces collisions but doesn't eliminate them.

CSMA/CD: Adds Collision Detection. While transmitting, station listens for collisions. If detected, aborts immediately, waits random backoff (Binary Exponential Backoff), retries.

🎤 Analogy: Walkie-talkie: listen first (CS), if collision (screech), stop, wait random time, retry.

🔐 3. Cryptography Basics | Symmetric Encryption & Block Cipher

Plaintext → Ciphertext • Encryption/Decryption • Shared Secret Key

📖 Explain

Cryptography: Science of secret writing. Encryption: Plaintext → Ciphertext using key. Decryption: Ciphertext → Plaintext.

Symmetric Key Cryptography: Same key used for encryption & decryption. Block Cipher: Encrypts fixed-size blocks (e.g., 64 bits) at a time (AES, DES).

🔑 Example: Caesar cipher (shift alphabet) is a simple symmetric cipher. Modern: AES-256.

📝 Exam-Style Questions & Answers (MCS-218 PYQ Pattern)

Q1. Give an expression for finding the throughput of the system in pure ALOHA. (Direct PYQ)
Answer: In Pure ALOHA, throughput S = G · e-2G, where G is the total arrival rate (load). Maximum throughput ≈ 18% at G = 0.5. Vulnerable period = 2 × frame transmission time.
Q2. Differentiate pure ALOHA with slotted ALOHA. (Direct PYQ)
Answer: Transmission: Pure ALOHA allows any time; Slotted ALOHA allows only at slot boundaries. Vulnerable period: Pure = 2T, Slotted = T. Efficiency: Pure ~18%, Slotted ~36%. Slotted reduces collision probability.
Q3. Explain the mechanism of CSMA/CD.
Answer: CSMA/CD: Station senses carrier before transmit. If idle, transmits while monitoring for collisions. If collision detected, aborts immediately, waits random backoff (Binary Exponential Backoff), then retries. Used in Ethernet.
Q4. Define cryptography, encryption, decryption, and block-cipher. (Direct PYQ)
Answer: Cryptography: Science of secure communication. Encryption: Plaintext → Ciphertext using key. Decryption: Ciphertext → Plaintext. Block Cipher: Symmetric cipher encrypting fixed-size blocks (e.g., AES).
Q5. How does cryptography contribute to network security? (Direct PYQ)
Answer: Cryptography provides: Confidentiality (data hidden), Authentication (identity verification), Integrity (no tampering), Non-repudiation (sender cannot deny). Used in SSL/TLS, VPNs, digital signatures.
