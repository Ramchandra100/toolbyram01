📍 1. K-Nearest Neighbors (K-NN) Algorithm

Lazy learner • Distance-based classification • Majority vote

📖 Explain

K-NN is a supervised classification algorithm that classifies new data points based on the majority class among its 'K' closest neighbors using Euclidean distance.

  • Steps: Choose K, calculate distances, find K nearest, majority vote, assign class.
  • Lazy Learner: No training phase; computes at prediction time.
🍎 Example: Mystery fruit with K=5 → 4 tomatoes, 1 apple → classify as Tomato!
📐 Euclidean Distance: d = √[(x₂-x₁)² + (y₂-y₁)²]

🌲 2. Decision Tree Classifier (J48 / C4.5)

Eager learner • Flowchart structure • Information Gain • Gini Index

📖 Explain

Decision Trees build a flowchart-like model by recursively splitting data based on attribute values. Uses Information Gain (based on Entropy) to select the best splitting attribute.

  • Root Node: Starting point (entire dataset)
  • Internal Nodes: Tests on attributes
  • Leaf Nodes: Final class labels
🏦 Loan Approval Example:
Root: "Has job?" → Yes → "Credit score > 700?" → Yes → Approve
→ No → Reject

📝 Exam-Style Questions & Answers (MCS-221 PYQ Pattern)

Q1. Write and explain the K-NN classification algorithm in Data Mining. (Direct PYQ - 10 Marks)
Answer: K-NN is a supervised classification algorithm. Steps: 1. Choose K. 2. Calculate Euclidean distance from new point to all training points. 3. Select K smallest distances. 4. Majority vote among K neighbors. 5. Assign class. It's a lazy learner — no training phase, computation at prediction time.
Q2. Discuss the advantages and disadvantages of the K-NN algorithm. (Direct PYQ)
Answer: Advantages: Simple, no training time, handles multi-class. Disadvantages: Slow prediction (computes all distances), sensitive to noise, requires large memory, performance degrades with high dimensions.
Q3. Write and explain the Decision Tree Classifier algorithm with the help of an example. (Direct PYQ - 10 Marks)
Answer: Decision Tree recursively splits data using attribute selection measures (Information Gain). Example: Play Tennis tree: Root = Outlook → Sunny branch tests Humidity, Overcast → Yes, Rain tests Wind. Each leaf gives final class. Interpretable and handles mixed data types.
Q4. Differentiate between "Eager Learners" and "Lazy Learners".
Answer: Eager Learners (Decision Trees): Build model during training, fast prediction, require training time. Lazy Learners (K-NN): No training, store all data, prediction is slow (computes at query time).
Q5. What is "Information Gain" in the context of Decision Trees?
Answer: Information Gain measures reduction in entropy after splitting on an attribute. Attributes with highest Information Gain are chosen as split nodes, creating the most efficient tree. Formula: IG = Entropy(parent) - Σ(weighted entropy of children).
