Part 1: Webmail vs. Email Client

Webmail
Browser-based (Gmail.com, Outlook.com)
Requires constant internet

Email Client
Dedicated app (Microsoft Outlook)
Downloads mail, works offline, syncs

At enterprises like IRICEN, officers use Microsoft Outlook. While auto-configuration often works, manual setup is essential when traveling, using home Wi-Fi, or non-standard devices.

Pro Insight: Automatic setup fails due to DNS or network restrictions — knowing manual settings makes you the hero.
Part 2: The Two "Postmen" — IMAP & SMTP

IMAP (Incoming)

Internet Message Access Protocol — delivers mail to your inbox, leaves original copy on server. Perfect sync across devices: phone, laptop, tablet all show same read/unread status.

Secure Port: 993 (SSL/TLS)
POP3 is older — downloads and removes from server. Rarely used today.

SMTP (Outgoing)

Simple Mail Transfer Protocol — takes your mail away to send to recipients. When you hit "Send," Outlook hands the message to SMTP server for routing.

Secure Port: 587 (STARTTLS)
Some legacy use port 465, but 587 is modern standard.
Part 3: The Secret Doors — Ports & Security

Servers have thousands of digital ports. Manual configuration requires exact port numbers, or the firewall blocks the connection.

Manual Configuration Example (Outlook / Mobile):
Incoming Server: imap.iricen.gov.in  |  Port: 993 (SSL/TLS)
Outgoing Server: smtp.iricen.gov.in  |  Port: 587 (STARTTLS)
Encryption ensures no one intercepts emails.
  • 1 Choose "Advanced Setup" / "Manual" in Outlook or Mail app
  • 2 Enter IMAP host & port 993, check SSL required
  • 3 Enter SMTP host & port 587, TLS/STARTTLS, authentication required
  • 4 Test account settings — if send fails, SMTP credentials need verification
    • Troubleshooting tip: If receiving works but sending fails, SMTP port or authentication is wrong — double-check port 587 and "My outgoing server requires authentication."

    Answer: Simple Mail Transfer Protocol — handles outgoing mail (sending).

    Q2: User wants email on office laptop AND mobile phone with read status synced. IMAP or POP3?

    Answer: IMAP — keeps master copy on server, syncs across devices.

    Answer: Port 993 — encrypted IMAP.

    Answer: Webmail = browser, always online. Email Client = installed app (Outlook), downloads emails, works offline.
