Theory & Field Guide

1. Ethernet Cable (RJ45)

Lifeline of the network: Connects PCs, printers, APs. The tiny plastic clip is everything — if broken, the cable slips and causes random disconnects. Always listen for a "CLICK".

Port LEDs: Green/Amber blinking = active data link. No light = no physical connection to switch or dead cable.

Golden Rule: If a user bumps the desk and loses internet → check RJ45 clip first.

2. Display Cables: VGA vs HDMI

VGA (blue, analog): Video only, needs screws tightened. Loose VGA = pink/yellow tint because color pins lose contact.

HDMI (modern flat connector): Carries high-quality video + audio in one cable. Common for projectors & smart boards.

3. Network Rack: Patch Panel + Switch

Patch Panel (passive): Terminates permanent wall cables. No electricity, just a safe bridge. Switch (active): Provides power & network intelligence. Short patch cords link panel to switch — saves expensive in-wall cables.

Always trace: Wall jack → Patch panel (back) → patch cord → Switch port.
