Part 1: Getting Inside the Brain (The Default Gateway)

Out of the box, a new Access Point is "dumb" — open, insecure network. Your job: log in and secure it.

How to log in (via web browser):
  1. Connect laptop to router (Ethernet is best for setup).
  2. Open Command Prompt → ipconfig → find Default Gateway (e.g., 192.168.1.1).
  3. Type that IP into Chrome address bar → press Enter.
C:\> ipconfig
Default Gateway . . . . . : 192.168.1.1
Navigate to http://192.168.1.1 → admin login page.
Factory default credentials: Sticker under device — typically admin / admin or admin / password. ⚠️ CHANGE IMMEDIATELY!
Part 2: The Two Golden Settings (SSID & Security)

SSID

Service Set Identifier = the Wi-Fi network name broadcasted to devices.

Security Protocol (Critical!)

WEP ❌ obsolete WPA ❌ outdated
✅ WPA2-PSK (AES) — current global standard
WPA3 — newest (modern routers)

Choose WPA2-PSK → enter strong password (e.g., Pune@2026) → Apply → router reboots.

Resident Engineer paycheck moment: Never leave network "Open". WPA2 encrypts all traffic, keeps parking lot hackers out.
Part 3: 2.4GHz vs 5GHz — Dual-Band Wisdom

📡 2.4 GHz

Long range — travels far through walls & concrete.
Slower speeds, more interference.
Ideal for coverage across building

⚡ 5 GHz

Extremely fast — low latency, high throughput.
Short range, struggles with thick walls.
Best in same room as AP

Pro Strategy: Modern routers broadcast both. Connect stationary devices to 5GHz for speed; mobile devices walking far use 2.4GHz for stability.

Quick Quiz: Level 5 (Wi-Fi Configuration)

Test your wireless expertise. Click any question to reveal the answer.

Q1: You run `ipconfig` to log into a new router. Which line tells you the router's IP address?

Answer: The Default Gateway line. That's the router's brain address.

Q2: What does SSID stand for, and what does it mean in plain English?

Answer: Service Set Identifier = the Wi-Fi network name broadcasted to users.

Q3: Which security protocol should you choose to ensure encrypted, safe Wi-Fi in a classroom?

Answer: WPA2-PSK (AES) — the global standard. WPA3 for newer gear.

Q4: Instructor's Wi-Fi keeps dropping when walking down the hall on 5GHz. What do you advise?

Answer: Switch to 2.4GHz — longer range, better wall penetration, more stable while moving.

Q5: Very first thing after logging into new router with default admin/admin?

Answer: Change the admin password! Factory defaults are a huge security risk — anyone on the network could hijack the router.
